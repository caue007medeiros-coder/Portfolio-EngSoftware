# Projetos Acadêmicos — CEUB

Este diretório armazena todos os projetos, algoritmos e atividades práticas desenvolvidos durante a minha graduação em Engenharia de Software no CEUB.

## 📚 Matérias e Conteúdos
* **Estrutura de Dados:** Algoritmos de ordenação, filas, pilhas e árvores.
* **Lógica de Programação:** Exercícios práticos desenvolvidos para fixação de conceitos básicos.
* **Engenharia de Requisitos:** Documentação, diagramas de caso de uso e histórias de usuário.

*Os códigos e resoluções serão commitados nesta pasta conforme o avanço dos semestres.*
